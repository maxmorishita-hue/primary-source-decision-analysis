# Primary-Source Decision Analysis (45-minute lesson)

This repository provides a classroom-ready lesson designed to help students
analyze historical decision-making using primary sources.

## Purpose
- To practice reading primary sources critically
- To identify constraints, assumptions, and decision inputs
- To explain outcomes without relying on moral judgment

## Target
- Upper secondary school students (advanced)
- Adaptable for middle school with simplification

## Key Features
- Uses authentic primary sources (Japanese & English)
- No fixed conclusion required
- Evaluation based on reasoning process, not opinion

## Lesson Length
- One class period (45 minutes)

## Topics Covered
- Diplomatic decision-making
- Constraints and limited options
- Interpretation of official documents

This material is intended for educational use and does not promote any
particular historical interpretation.
